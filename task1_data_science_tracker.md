# Task 1: Data Science Tracker

## 目标
创建一个数据科学项目跟踪器，用于监控和管理机器学习实验。

## 要求
1. 记录实验名称、数据集、模型类型、准确率、F1分数
2. 支持添加新实验
3. 支持查询实验（按准确率排序）
4. 计算平均准确率
5. 导出实验数据为CSV格式

## 预期行为
- 添加实验时，检查必要字段是否完整
- 查询时返回排序后的实验列表
- 平均准确率计算正确（精度保留2位小数）
- CSV导出文件包含表头

## 边界条件
- 空实验列表时，平均值返回0.00
- 重复实验名称应拒绝添加或更新（根据实现）
- 数据集名称不应为空字符串
- 准确率应在0-1范围内
